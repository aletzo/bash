This is a simple script to set up my terminal environment settings (`.bash_profile` and `.zshrc`).

Additional scripts are inside the `.shellscripts` directory.

HOW TO INSTALL
==============

1. cd ~
2. git clone http://github.com/aletzo/bash.git ~/dotfiles
3. cd dotfiles
4. sh bootstrap.sh

HOW TO UPDATE
=============
1. cd ~/dotfiles
2. sh bootstrap.sh
